1. Select the project from the dropdown,upload the JSON file and submit it to mock the service.
2. If the project is not listed in the dropdown, go ahead and add project.
3. The Json file which is uploaded should have the same name as that of service(API).
4. While proxy passing the request in NGINX, URL will be in the following format http://{ip_of_node_server}:{port}/{projectname}

Exp: http://10.96.90.204:8000/NAC.
5. The following entries should be made in NGINX conf file

location ^~ /api/ {
proxy_pass http://10.96.90.204:8000/NAC/;
}


location ^~ /v1/ {
proxy_pass http://10.96.90.204:8000/NAC/v1/;
}
